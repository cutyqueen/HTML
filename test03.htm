웹에서의 색상

빛의 삼원색(red,green,blue)
각 색상별 0~255까지 값을 지정.
16진수로 색상을 표시할 때는 (0~9,a,b,...,f)

  1. red = rgb(255,0,0) = #ff0000
  2. green = rgb(0,255,0) = #00ff00
  3. blue = rgb(0,0,255) = #0000ff
  4. yellow = rgb(255,255,0) = #ffff00
  5. white = rgb(255,255,255) = #ffffff
  6. black = rgb(0,0,0) = #000000
