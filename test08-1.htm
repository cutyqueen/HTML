6. input type="tel" - 연락처(전화번호)를 지정하는 요소

모바일브라우저에서 지원하는 경우 가상키패드가 숫자형식으로 바뀌고 전화걸기 가능

7. input type="email" - 연락처(이메일)를 지정하는 요소

브라우저에서 알맞은 이메일형식인지 확인해주고(@유무 등) 모바일브라우저에서 지원하는 경우 가상키패드에 @, .com 등이 나올 수 있다.

8. input type="url" - 연락처(웹주소)를 지정하는 요소

브라우저에서 알맞은 주소형식인지 확인해주고(프로토콜 https:// 유무) 모바일브라우저에서 지원하는 경우 가상키패드에 .com 등이 나올 수 있다.

11. input type="color" - 색상값을 지정하는 요소

12. input type="search" - 검색용 창을 지정하는 요소

13. input type="hidden" - 사용자에게 보이지않는 숨은값 지정하는 요소

fieldset과 legend로 폼요소들을 그룹화

[ 속성 ]

1. aufocus(autofocus="autofocus") - 자동 포커싱

2. autocomplete="on/off" 자동완성 기능을 켜고 끔

3. placeholder="..." 입력예시나 설명 등을 추가

4. required(required="required") - 필수입력으로 지정

5. readonly(readonly="readonly") - 읽기만 가능한 요소로 지정 (전송 됨)

6. disabled(disabled="disabled") - 사용불가능한 요소로 지정 (전송 안됨)
