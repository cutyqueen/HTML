input으로 시작하는 폼요소들

input에 대한 설명문을 label태그로 지정하고 서로 연결해주려면
label에는 for의 값을, input에는 id의 값을 서로 일치시킴.

1. input type="text" 주관식. 한 줄 글상자

최소값을 minlength="n"로, 최대값을 maxlength="n"로 지정 가능.
상자 길이는 size="n"로 지정 가능.

2. input type="password" 주관식. 암호 글상자

3. textarea ... /textarea 주관식. 큰 글상자

가로 크기는 cols="n" 세로 크기는 rows="n"로 지정

4. input type="checkbox" 다중선택형 객관식. 체크박스

체크박스과 라디오버튼은 초기값으로 지정할 항목에 checked(checked="checked")를 추가함.

5. input type="radio" 단일선택형 객관식. 라디오버튼

단일선택형이 되려면 선택지들끼리 name의 값을 일치시킴

6. input type="submit/reset/button" 전송/취소/일반 버튼


6-1. button ... /button 전송용 버튼(취소/일반)

type="reset/button"을 추가하여 취소나 일반버튼으로 변경 가능

7. input type="image" 전송용 이미지

이미지 경로는 src로 지정

8. input type="file" 파일 전송용
